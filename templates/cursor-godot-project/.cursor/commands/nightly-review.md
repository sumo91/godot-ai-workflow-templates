Review the result of an overnight Godot run before continuing work.

$ARGUMENTS

Process:

1. Read the overnight summary.
2. Summarize the actual change scope.
3. Check whether any high-risk files were touched.
4. Identify what validation already ran and what is still missing.
5. Recommend whether to continue, re-validate, re-plan, or revert.

Output:

- Overnight Outcome
- Changed Scope
- High-Risk File Status
- Validation Status
- Open Risks
- Recommended Next Step
