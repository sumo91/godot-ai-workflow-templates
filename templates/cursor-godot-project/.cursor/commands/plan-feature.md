Turn this Godot feature idea into a low-conflict implementation plan.

$ARGUMENTS

Requirements:

- Start from an approved or clearly restated design.
- Split the work into small task packets.
- For each task packet include:
  - Task ID
  - Goal
  - Allowed Files
  - Forbidden Files
  - Expected Changes
  - Dependencies
  - Validation Steps
  - Definition of Done

Also include:

- Shared files
- Parallel-safe tasks
- Serial-only tasks
- Recommended execution order
- Which work must stay in the main session
