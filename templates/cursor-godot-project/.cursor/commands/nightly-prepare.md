Prepare an overnight Godot task file for Cursor Background Agents or local nightly automation.

$ARGUMENTS

Create or refine a single overnight task with:

- Goal
- Context
- Allowed Files
- Forbidden Files
- Validation
- Definition of Done
- Recommended Mode: `safe`, `low-risk`, or `task-packet`

Rules:

- overnight work must be task-file driven
- high-risk shared-file integration stays in the main session
- if ownership is unclear, prefer `safe`

Output:

- Nightly Goal
- Recommended Mode
- Allowed Files
- Forbidden Files
- Validation Plan
- Definition of Done
- Suggested First Step For Tomorrow
