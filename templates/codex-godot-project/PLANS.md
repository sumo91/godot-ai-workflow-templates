# Plans Guide

For medium or large tasks in this project:

1. Write a minimal spec before implementation.
2. Break the work into task packets.
3. Keep shared-file integration in the main session.
4. Prefer one task packet per execution thread.
5. Run validation before saying the work is done.

Recommended task packet fields:

- Goal
- Context
- Allowed Files
- Forbidden Files
- Validation
- Done Criteria
