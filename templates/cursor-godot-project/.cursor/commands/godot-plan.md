Use the planning workflow to turn an approved Godot design into task packets.

$ARGUMENTS

Requirements:

- confirm the planning source
- split the work into low-conflict task packets
- mark which tasks are parallel-safe
- mark which tasks must stay in the main session
- define allowed files, forbidden files, and validation for each task

Output:

- Planning Source
- Shared Files
- Parallel-Safe Tasks
- Serial-Only Tasks
- Recommended Execution Order
- Task Packets
