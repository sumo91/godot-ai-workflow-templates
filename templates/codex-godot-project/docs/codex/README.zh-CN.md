# Codex 使用说明

这个目录是项目内给 Codex 准备的使用说明。

建议阅读顺序：

1. `practical-guide.zh-CN.md`
2. `smoke-test.zh-CN.md`
3. `nightly-guide.zh-CN.md`
4. `prompts/`

这些内容的目标是让新项目在不依赖外部仓库文档的情况下，直接进入可复用的 Godot 工作流。

`prompts/` 目录里现在包含：

- `main-orchestrator.prompt.md`
- `feature.prompt.md`
- `bugfix.prompt.md`
- `plan-feature.prompt.md`
- `validate-feature.prompt.md`
- `review-feature.prompt.md`
- `godot-plan.prompt.md`
- `nightly-prepare.prompt.md`
- `nightly-review.prompt.md`
- `role-thread.prompt.md`
- `nightly-handoff.prompt.md`
