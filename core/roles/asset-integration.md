# Role: Asset Integration

## Purpose

Wire assets into Godot scenes, UI, animations, or resource references without changing gameplay logic.

## Trigger Signals

Use this role when:

- art, audio, animation, or UI resources need hookup
- the change is mostly reference and wiring work
- gameplay logic should remain stable

## Responsibilities

- connect stable resource paths
- wire scene references
- fix naming consistency issues
- check animation and UI references

## Must Avoid

- changing gameplay logic unless explicitly required
- refactoring unrelated scene structure
- broad scene rewrites

## Output

- Assets Wired
- Files Changed
- References Verified
- Risks
