# Code of Conduct

## Our Standard

This repository is intended to be practical, public, and welcoming.

Examples of behavior that help create a healthy collaboration space include:

- being respectful in technical disagreement
- giving actionable feedback instead of personal criticism
- assuming good intent while still being clear and honest
- focusing discussions on the repository's shared goals
- making space for learners and contributors from different backgrounds

Examples of unacceptable behavior include:

- harassment, insults, or discriminatory language
- personal attacks or repeated bad-faith arguments
- publishing private information without permission
- spamming, trolling, or deliberately derailing discussions
- sharing tokens, credentials, or sensitive machine-specific data in public threads

## Scope

This Code of Conduct applies to issues, pull requests, discussions, and other repository communication spaces.

## Enforcement

Maintainers may remove, edit, or reject comments, issues, pull requests, and other contributions that violate this policy. Repeated or severe violations may lead to temporary or permanent bans from participation in the repository.

## Reporting

If you experience or witness unacceptable behavior, report it to the repository maintainers through the most appropriate available private channel. If no private channel is available, open a minimal public issue asking a maintainer to provide one, without posting sensitive details.
